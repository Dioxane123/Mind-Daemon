# Mind Daemon
**目标人群**：所有想要完成某些任务的人群，尤其是容易分心的ADHD患者。
**目标简述**：

- 最开始让用户设定一个一段时间内的总目标和预计完成时间。
- 随后利用BCI设备检测用户此时的专注程度。
    - 当用户的专注程度低于一定程度时，系统进入放松状态，采取措施让用户进入放松时间。
    - 用户觉得休息时间足够或者到了指定休息时间时屏幕边缘出现光晕软性提醒用户，用户通过摄像头手势让系统进入准备专注状态。
    - 准备专注状态播放背景音乐等等方式辅助用户专注。当BCI设备检测到用户的专注程度高于一定程度时认为进入心流状态，逐渐停止背景音乐等等。
    - 当用户的专注时间达到自己的设定值或专注程度又低于一定程度时，系统再次进入放松状态。
    - 在检测到用户瘫倒在椅子上时认为进入长期休息状态，电脑自动锁屏。
